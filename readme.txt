A widget which helps the users to display the quote of the day in any WordPress website.


== Description ==

The Daily Quotes widget helps the users to display the quote of the day on the WordPress websites. 

Major features in Daily Quotes include:

* User can give a title to the widget section displayed.
* The quote is displayed from the quotery.com site.
* User can define the border color for the widget display.


== Installation ==

**Follow the steps to install the plugin:**

1. Navigate to Add New Plugin page within your WordPress
2. Search for Daily Quotes Plugin
3. Click Install Now link on the plugin and follow the prompts
4. Activate the plugin through the 'Plugins' menu in WordPress

Or

1. You can also download the plugin from the following location:
http://bobcares.com/software-and-design-services/wordpress-plugins/daily-quotes.html
2. Upload plugin directory to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress

After the plugin activation, go to Appearance &gt;&gt Widgets and drag the Daily Quotes widget to the widget area.
Enter the title and border color and click the save button and visit the website.


== Frequently Asked Questions ==  

= Can I use this plugin in any WordPress site? =

Yes, you can use this plugin in any WordPress site.

= From where the quote gets displayed? =

The quote is fetched from the site quotery.com.

= How can I set the title and border color for the widget? =

After activating the plugin, go to Appearance &gt;&gt; Widgets and drag the Daily Quotes widget to the widget area.
Enter the title which you need to display on the site. Also, there is an option for entering the border color.
After entering both options, just click save and visit your site.
